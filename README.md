# 🏦 Streamlit Banking System

A **mini banking web application** built using **Python and Streamlit** that allows users to perform basic banking operations through a simple web interface. The application stores user data using a **JSON-based database**.

## 🚀 Features

* Create a new bank account
* Deposit money
* Withdraw money
* View account details
* Update user information
* Delete account
* Persistent data storage using JSON

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **JSON (for database)**
* **Git & GitHub**

## 📂 Project Structure

```
streamlit-banking-system
│
├── app.py            # Streamlit frontend
├── bank.py           # Banking system logic
├── data.json         # JSON database
└── README.md         # Project documentation
```

## ⚙️ Installation & Setup

1. **Clone the repository**

```
git clone https://github.com/Anu19-10/streamlit-banking-system.git
```

2. **Navigate to the project folder**

```
cd streamlit-banking-system
```

3. **Install dependencies**

```
pip install -r requirements.txt
```

4. **Run the Streamlit application**

```
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```

## 📸 How It Works

1. Users can create an account by providing name, age, email, and PIN.
2. Each account receives a **unique account number**.
3. Users can securely deposit and withdraw money using their account credentials.
4. Account data is stored in a **JSON file** to maintain persistence.
